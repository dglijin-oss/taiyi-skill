# 太乙神数技能

> "太乙通神，国运可测。" —— 传统术数

太乙神数是中国古代最高层次的预测术数之一，与奇门遁甲、大六壬合称"三式"。太乙主要用于国运预测、天灾预警、经济大势、战争胜负等宏观预测。本技能提供完整的太乙排局和断局功能。

## ✨ 核心功能

- ✨ 太乙积年计算
- ✨ 起局推算（阳遁/阴遁）
- ✨ 十六神将排布
- ✨ 主客算计算
- ✨ 五福十神推算
- ✨ 五行生克分析
- ✨ 自动化断局（吉凶判断、断语）
- ✨ 国运/天灾/经济/战争预测
- ✨ 纯 JavaScript 实现，无外部依赖

## 📖 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dglijin-oss/taiyi-skill.git
cd taiyi-skill

# 确保 Node.js 14+ 已安装
node --version

# 赋予执行权限
chmod +x index.js
```

### 基本使用

```bash
# 当前时间起局
node index.js

# 指定时间起局
node index.js --date "2026-04-10 15:00"

# 指定问事类型（国运/天灾/经济/战争）
node index.js --date "2026-04-10 15:00" --question 国运

# 输出 JSON 格式
node index.js --date "2026-04-10 15:00" --json
```

### 命令行参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| --date | -d | 日期时间 (YYYY-MM-DD HH:MM) | 当前时间 |
| --question | -q | 问事类型 | 国运 |
| --json | -j | 输出 JSON 格式 | 否 |

### 问事类型

| 类型 | 说明 |
|------|------|
| 国运 | 国家运势、政局稳定 |
| 天灾 | 自然灾害预警 |
| 经济 | 经济形势、市场走势 |
| 战争 | 战争胜负、军事冲突 |
| 通用 | 综合判断 |

## 📚 太乙神数基础知识

### 太乙积年

太乙积年是从上元甲子到当前年份的累积年数，用于起局计算。

计算公式：
```
积年 = (当前年份 - 上元年份) + 基数
```

### 阴阳遁

| 遁法 | 使用季节 | 行法 |
|------|----------|------|
| 阳遁 | 冬至 - 夏至 | 顺行 |
| 阴遁 | 夏至 - 冬至 | 逆行 |

### 十六神将

| 神将 | 含义 |
|------|------|
| 太乙 | 主神，代表核心 |
| 文昌 | 文运、文化、教育 |
| 主算 | 主方运势 |
| 客算 | 客方运势 |
| 始击 | 冲击、变动 |
| 天目 | 洞察、预见 |
| 地目 | 地理、环境 |
| 飞鸟 | 快速、消息 |
| 直符 | 领导、权威 |
| 腾蛇 | 变化、虚惊 |
| 朱雀 | 口舌、文书 |
| 六合 | 和合、交易 |
| 勾陈 | 争斗、诉讼 |
| 青龙 | 喜庆、财运 |
| 白虎 | 血光、疾病 |
| 玄武 | 盗贼、欺骗 |

### 主客算

- 主算：代表主方、内部、守势
- 客算：代表客方、外部、攻势
- 主算 > 客算：主胜
- 客算 > 主算：客胜
- 主算 = 客算：平局
