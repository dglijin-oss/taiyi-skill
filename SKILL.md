## 3. How to use

### 调用方式

```bash
# 基础排局（当前时间）
python3 ~/.openclaw/skills/taiyi-skill/scripts/taiyi_pan.py

# 指定时间排局
python3 ~/.openclaw/skills/taiyi-skill/scripts/taiyi_pan.py --date "2026-04-03"

# 指定问事类型
python3 ~/.openclaw/skills/taiyi-skill/scripts/taiyi_pan.py --date "2026-04-03" --question "国运"

# 输出 JSON 格式
python3 ~/.openclaw/skills/taiyi-skill/scripts/taiyi_pan.py --date "2026-04-03" --json
```

### 命令行参数

| 参数 | 简写 | 说明 |
|------|------|------|
| --date | -d | 占问时间 (YYYY-MM-DD) |
| --question | -q | 问事类型（国运/天灾/经济/战争/选举/疫情/行业） |
| --json | -j | 输出 JSON 格式 |
